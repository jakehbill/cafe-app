# Beaned marketing site

Next.js marketing surface for [beaned.app](https://web.beaned.app). Separate from the Expo app so we can use Tailwind and Framer Motion without coupling to React Native Web.

## Develop

```bash
cd marketing
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js (App Router)
- Tailwind CSS
- Framer Motion
